About page

This website provides the current and past job opportunities at the City of Ottawa. The data obtained comes directly from the Ottawa Open data portal. All the jobs are in ascending order on the Expiration Date. Meaning that Jobs that will be expired soon will be first and jobs that will be expired later will be further down the pagination.

You can find the github repository of this web application here. This project is inpired from OttawaCityJobs

The content of this website is under the CC-BY-ND license. The images of the Ottawa city flags is under the Wiki Creative Commons License. The code for this application is also available on Github. Some third party of the code can't be transfert like the Twitter bot api and the bitly api.

À Propos

Cette page web fournit la plus récente liste d'emplois à Ville d'Ottawa. Les emplois qui ne sont plus affichés sur le site web de la ville parce qu'ils ont été expirés, sont aussi accessible sur ce site. Les données ont été récupérés sur le portail de données ouvertes d'Ottawa.Les offres d'emploi sont listés en ordre croissant en prenant comme référence la date d'expiration. Ce qui veut dire que les emplois qui expiront bientôt seront les premiers à être affichés.

Vous pourrez trouvez les scripts nécessaires à la création de cette application web en cliquant ici. Ce projet à été inspiré de cette application web OttawaCityJobs.

Le contenu de ce site web est sous license CC-BY-ND. L'image du drapeau de la ville d'Ottawa est sous license Wiki Creative Commons. Les scripts qui ont permis de créer cette application web sont aussi disponible sur Github sous license CC-BY. Les scripts contenants des informations personnelles par exemple pour faire fonctionner la publication de Tweet sur Twitter n'ont pas été inclus dans le code sur Github
